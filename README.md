Mod de Stardew Valley que introduce a los personajes de la web-serie Eddsworld; Edd, Matt, Tord y Tom, así como una nueva localización y eventos de corazones nuevos.

<p align="center">
  <img src="https://github.com/Martiitale/EddsValley/blob/main/GUIDE%20for%20his%20likes%20and%20dislikes/Installation.png?raw=true">
</p>

1. Descargar [Smapi](https://smapi.io/) (Ver instrucciones de instalación.)  
2. Descargar [ContentPatcher](https://www.nexusmods.com/stardewvalley/mods/1915) (Ver instrucciones de instalación.)  
3. Descargar archivo EddsValley.zip  
4. Descomprimir EddsValley.zip en la carpeta de mods creada por Smapi
